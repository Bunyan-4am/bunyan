## ğŸš¨ Troubleshooting

### "Using demo mode" message appears
**Cause:** Gemini API key not configured or invalid
**Fix:** Check `.env.local` file has correct `GOOGLE_GEMINI_API_KEY`

### Image doesn't upload
**Causes:**
- File too large (>10MB)
- Unsupported format
- Browser issue

**Fix:**
- Compress image
- Convert to JPG/PNG
- Try different browser

### AI response is generic
**Causes:**
- Vague question
- No context provided
- Image quality poor

**Fix:**
- Be more specific
- Add project details
- Upload clearer images

### Generated image doesn't load
**Cause:** Free image service might be slow
**Fix:** 
- Wait a few seconds
- Refresh page
- Image URL still works for viewing

### "Failed to scan bill" error
**Causes:**
- Not a bill/invoice
- Text not readable
- Wrong language

**Fix:**
- Upload actual invoice
- Ensure text is clear
- Use English/Arabic invoices
